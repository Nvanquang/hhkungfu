## Ghi Chú Thiết Kế

### Layout Auth
- Tất cả trang auth dùng **centered card layout** — card trắng trên nền dark gradient hoặc ảnh anime mờ
- Card width: max `w-md` (448px) trên desktop, full width `px-4` trên mobile
- Logo Hhkungfu ở đầu card, link về `/`

### OTP Input
- 6 ô riêng biệt, mỗi ô `w-12 h-14`, tự động focus ô tiếp theo khi nhập
- Hỗ trợ paste chuỗi 6 số vào ô đầu → tự điền tất cả
- Border highlight màu primary khi focus, đỏ khi error

### Password Strength Bar
- 4 mức: Yếu (đỏ 25%) · Trung bình (vàng 50%) · Tốt (xanh 75%) · Mạnh (xanh đậm 100%)
- Hiện khi user bắt đầu gõ password

### Error States
- Field error: border đỏ + text đỏ nhỏ bên dưới
- Global error (credentials sai): banner đỏ nằm trên button
- Success: banner xanh hoặc redirect

### Google OAuth
- Sau OAuth callback: `GET /oauth/callback?token=xxx` → FE lưu token → redirect về `/`
- Nếu email đã đăng ký LOCAL → hiện lỗi riêng trên trang login

### Banner "chưa xác thực email"
- Hiện sticky ở top của mọi trang sau khi đăng nhập nếu `emailVerified = false`
- Màu vàng, có nút [Xác thực ngay] → `/verify-email`
- Có thể dismiss tạm thời (localStorage) nhưng hiện lại sau 24h
