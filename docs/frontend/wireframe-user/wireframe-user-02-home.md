## Ghi Chú Thiết Kế

### Navbar
- Desktop: full navbar ngang, logo trái — links giữa — actions phải
- Mobile: chỉ logo + icon search + icon user (text ẩn), bottom nav cho navigation chính
- Sticky + `backdrop-blur` để nhìn xuyên khi cuộn

### Hero Section
- Auto-slide sau 6 giây, dots để navigate thủ công
- Gradient overlay từ phải sang trái (ảnh mờ bên phải, nội dung rõ bên trái)
- Mobile: thumbnail ẩn, chỉ hiện title + info + buttons trên banner

### Horizontal Scroll Rows
- CSS `overflow-x: auto`, scroll-snap, `scrollbar-hidden`
- Chevron `<` `>` hiện khi hover (desktop only)
- "Tiếp tục xem" có progress bar trên mỗi card (màu primary, h-1)

### Genre Chip Filter
- Click chip → filter anime bên dưới (không navigate trang)
- Active chip: filled background màu primary
- Thanh chip cuộn ngang trên mobile

### VIP Banner CTA
- Ẩn hoàn toàn nếu `user.isVip = true`
- Gradient tím → vàng, có animation shimmer nhẹ
