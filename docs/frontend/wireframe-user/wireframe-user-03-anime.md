## Ghi Chú Thiết Kế

### Anime Card
- Thumbnail dùng tỷ lệ `2:3` (poster đứng), hover → overlay tối nhẹ + icon play
- Badge trạng thái góc trên trái: `ONGOING` xanh lá · `COMPLETED` xám · `UPCOMING` vàng
- Badge VIP góc trên phải: gradient vàng/tím `✨ VIP`
- Hover card (desktop): scale 1.03, shadow lớn hơn, transition 150ms

### Filter Bar (Desktop)
- Dropdowns dạng `<select>` custom có search bên trong
- Active filter → nút "Xóa bộ lọc" hiện ra
- Thay đổi filter → re-fetch, giữ params trên URL (`?status=ONGOING&genre=action`)

### Trang Detail — Tab "Tập phim"
- Mặc định hiển thị danh sách dọc, mỗi hàng = 1 tập
- Tập đang xem dở: highlight + hiển thị progress bar ngang
- Tập VIP-only (khi chưa có VIP): lock icon + text "Nâng VIP để xem" + nút CTA
- Cuộn đến tập đang xem dở khi mở tab lần đầu

### Search — Instant Suggestions
- Debounce 300ms khi gõ → gọi Elasticsearch API
- Tối đa 5 gợi ý trong dropdown
- Lưu 5 từ khóa gần nhất vào `localStorage` → hiện khi focus ô trống
- Kết quả trang đầy đủ: list view (không grid) để hiện description

### Rating Stars
- Hover từng sao → fill tạm thời
- Click → POST /ratings, optimistic update
- Nếu đã đánh giá → sao filled + tooltip "Bạn đánh giá X/10. Click để thay đổi"
