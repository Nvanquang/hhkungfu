## Ghi Chú Thiết Kế

### Trang Profile Công Khai
- Chỉ hiện lịch sử tập đã xem **hoàn thành** (không lộ tập đang xem dở)
- Không có section bookmark trên profile công khai của người khác
- Nếu xem profile của chính mình → hiện nút [⚙ Chỉnh sửa]

### Lịch Sử Xem
- Nhóm theo ngày: Hôm nay · Hôm qua · Tuần này · Tháng này
- Mỗi item: thumbnail 16:9 · tên anime · tên tập · progress bar · thời gian
- Xóa từng item: long-press (mobile) hoặc hover hiện nút [✕]
- Xóa toàn bộ: dialog confirm trước khi xóa

### Bookmark
- Hover card: hiện overlay với nút [✕ Bỏ bookmark] (desktop)
- Mobile: long-press → menu "Bỏ bookmark"
- Click card → navigate về trang detail anime

### Cài Đặt
- Desktop: sidebar layout 2 cột
- Mobile: menu list → navigate sang trang con từng mục
- Email field: read-only (lock icon), không cho đổi email
- Tài khoản Google OAuth: ẩn section "Đổi mật khẩu", thay bằng "Tài khoản Google"
- Lưu thay đổi: optimistic update + toast success/error

### Bottom Navigation (Mobile)
```
🏠 Home     → /
🔍 Search   → /search
📚 Library  → /me/bookmarks  (hoặc /login nếu chưa đăng nhập)
👤 Profile  → /me/settings   (hoặc /login)
```
Active tab: icon + text đổi màu primary, underline indicator
